# TIL (Today I Learned)

학습한 내용을 정리하고 기록하는 저장소입니다.

## 📂 구성 및 운영 원칙

### 1. 이론 및 학습 정리
- 소스코드 외에 이론 공부 등 학습한 내용을 정리하여 업로드합니다.

### 2. 과거 기록 (AI 가독성 개선)
- 과거에 공부했던 내용들은 **생성형 AI**를 활용하여 내용은 그대로 유지하되, 가독성을 높인 마크다운(md) 파일로 변환하여 저장합니다.
- 사용한 명령어는 "내용 변경은 일절없이 디자인만 개선하여 가독성을 높여서 마트다운 파일 형식으로 제공하여라."

### 3. 신규 기록 (직접 작성)
- 새롭게 생성하는 마크다운 파일 역시 AI를 디자인 개선으로만 사용하거나 혹은 직접 필기하였다면 md외의 pdf 혹은 이미지로도 저장합니다.
